# official-home-notes

本仓库用于归档和发布多个独立的 HTML 页面。这些页面以静态文件形式存放，适合通过 GitHub Pages 或其他静态托管服务直接访问。

## 目录结构

```
├── index.html           # 仓库首页（可选）
├── pages/               # 存放各个 HTML 页面
│   ├── example-1.html
│   ├── example-2.html
│   └── ...
├── assets/              # 页面引用的资源文件（CSS、JS、图片等）
│   ├── css/
│   ├── js/
│   └── images/
└── README.md            # 本说明文件
```

## 页面归档说明

- 每个 HTML 页面均为独立完整的文件，可直接在浏览器中打开。
- 页面之间无强依赖关系，可以单独更新或删除。
- 如需添加新页面，请在 `pages/` 目录下创建 `.html` 文件，并确保引用资源路径正确。

## 维护说明

- 本仓库主要用于个人或团队内部归档，不对外提供具体服务承诺。
- 页面内容可能不定期更新，恕不另行通知。
- 如发现页面显示异常，请检查资源文件路径或浏览器兼容性。
- 欢迎提交 Issue 或 Pull Request 以协助改进内容或结构。

## 使用方式

1. 克隆或下载本仓库到本地。
2. 使用任意静态服务器（如 `python -m http.server`）在项目根目录启动服务。
3. 在浏览器中访问 `http://localhost:8000` 即可查看页面列表。

本仓库内容仅供参考和学习使用，不构成任何形式的保证。
